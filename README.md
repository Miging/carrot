# 당근마켓 주요기능

1. 홈 (/)
    - 가져오는 정보들
    - 글 제목
    - 위치
    - 가격
    - 사진
    - 채팅 수
    - 하트 수 
2. 상세페이지 조회(/{id})
    - 상세페이지 가져와야할 정보들
        - 사용자
            -  이름
            -  매너온도
            -  위치
        - 글
            - 제목
            - 내용
            - 관심수
            - 조회수
            - 채팅수
            - 가격
            - 첨부 이미지(다중 이미지)
                - 가격 흥정 가능 여부
            - 판매 완료 여부
3. Response 형식
```
{
	"suceess": true,
	"msg": "요청에 성공했습니다."
	"data": {
		"key1": "블라블라 데이터",
		"key2": 150000,
		"key3": "블라블라 값"
	}
}
```
